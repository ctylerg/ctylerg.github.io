**Instructions**
Sign-in and post texts or pictures of random acts of kindness you do or see.


** Technologies Used:**

 - Ruby on Rails
 - Javascript
 - Postgresql database
 - jQuery for DOM manipulation
 - Base 64
 - Github
 - Ajax
 - CRUD
 - Skeleton to refine the overall look of the app
 - Geomaps
