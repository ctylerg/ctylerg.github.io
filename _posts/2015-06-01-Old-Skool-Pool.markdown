**Instructions**
Guess which year the Men's NCAA tournament occurred before time runs out.


** Technologies Used:**

 - Sinatra
 - Javascript
 - Postgresql database
 - Ruby
 - jQuery for DOM manipulation
 - nokogiri to scrape data
 - Github
 - Ajax
 - CRUD
 - Bootstrap to refine the overall look of the app
