ABOUT THE PROJECT

This Sinatra app incorporated a lot of Javascript and jQuery to get the data to render in a usable fashion. To get the data for this app, it had to be scraped using the 'nokogiri' gem. Once the data was digestible it was entered into the postgreSQL database. API calls were made to the database to retrieve the necessary data.
